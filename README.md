# jobtailor

Stop rewriting your CV for every job. jobtailor tailors it for you, in the job's language, and keeps everything on your computer.

## What you get

* A tailored CV and cover letter (.docx).
* An ATS keyword score.
* Files saved by date in a folder you choose.
* Privacy note: your CV and keys stay on your machine, nothing is uploaded.

## Getting started (no terminal)

1. Install Python from https://www.python.org/downloads/ (check "Add Python to PATH" on Windows).
2. On GitHub, click the green Code button, then Download ZIP.
3. Unzip the download.
4. Double-click run.bat (Windows) or run.command (Mac).

Note: The app runs free with a Groq key (https://console.groq.com/keys). It is French by default with an English toggle, and the CV and letter come out in the offer's language.
